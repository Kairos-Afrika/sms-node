A kairos afrika NPM wrapper that exposes all the apis making integration into your backend for sending SMS easier and faster

## Usage
### Installation
Quickly use the package by running either of the below commands
```bash
npm i @kairos/sms
```
or
```bash
yarn add @kairos/sms
```
